# Display 與 Live 動畫規則

## 時間與常數

- **x**：一張便利貼在 Display 上「完全靜止」的秒數。
- **過場動畫**：**x×0.1** 秒，Display 與 Live 共用這段時間、同時動。
- **反推**：總時長 = x + x×0.1 ⇒ **x = DISPLAY_SLOT_DURATION_SECONDS / 1.1**，過場 = **x×0.1**；過場前半 = 後半 = **x×0.05**。

---

## 規則一：Idle 交換（A → B，無新 pending）

**第一次 x×0.1 過場**

- **前半**
  - Display：A 從 Display 出場到「畫面左邊外、中間」。
  - Live：B 從「原本貼的位置」出場到「畫面右邊外、中間」。
- **後半**
  - Display：B 從「畫面左邊外、中間」進場。
  - Live：A 從「畫面右邊中間」進場，貼到 **B 原本的位置**。

之後 B 在 Display 不動 x 秒。

---

## 規則二：Idle 交換（B → C）

**第二次 x×0.1 過場**

- **前半**
  - Display：B 往左出場。
  - Live：C 從牆上往右出場。
- **後半**
  - Display：C 從左進場。
  - Live：B 從右進場，貼到 **C 原本的位置**。

之後 C 在 Display 不動 x 秒。

---

## 規則三：新便利貼 G 在 C 靜止期間加入（C 滿 x 秒後：C 出、G 進）

前提：C 在 Display 不動 x 秒的這段時間，有人新增便利貼 G。

**第三次 x×0.1 過場**

- **Display**
  - 前半：C 從 Display 出場到「畫面左邊外、中間」。
  - 後半：**G** 從「畫面下面外、中間」進場（新 pending）。
- **Live**
  - 前半：不需要有動作（沒有人從 Live 飛去 Display）。
  - 後半：**C** 從「畫面右邊中間」進場，貼回 **C 原本的位置**（此時牆是滿的，沒有人被擠出）。

---

## 規則四：G 不動 x 秒後，下一張是 D（從 Live 借，牆已滿 → 擠掉最舊）

**第四次 x×0.1 過場**

- **Display**
  - 前半：G 從 Display 出場到「畫面左邊外、中間」。
  - 後半：**D** 從「畫面左邊外、中間」進場（從 Live 借的，從左進場）。
- **Live**
  - 前半（同時）：
    - (1) 最舊一張從原本位置出場到「畫面上方外、中間」；
    - (2) D 從原本位置出場到「畫面右邊外、中間」。
  - 後半：**G** 從「畫面右邊中間」進場，貼到 **D 原本的位置**。

---

## 以 B 為例的一生

- 第一次 x×0.1 前半：在 Live 出場（往右）。
- 第一次 x×0.1 後半：在 Display 進場（從左）。
- B 在 Display 不動 x 秒。
- 第二次 x×0.1 前半：在 Display 出場（往左）。
- 第二次 x×0.1 後半：在 Live 進場（從右，貼到 C 原本的位置）。
